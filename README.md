# serverless-demo

## Setup

```bash
npm install serverless
```

Create a [IAM role and access key](https://www.serverless.com/framework/docs/providers/aws/guide/credentials/).

## Usage

Create the project:

```
serverless
```

Deploy the project (with verbose logging):

```
serverless deploy -v
```

Local execution:

```
serverless invoke -f demo
```
